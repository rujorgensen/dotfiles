# Install tools

## VS Code

Download the .deb-package:
`wget -O vscode.deb "https://code.visualstudio.com/sha/download?build=stable&os=linux-deb-x64"`

Install:
`sudo apt install ./vscode.deb -y`

Remove the package:
`rm vscode.deb`

## Lazydocker
